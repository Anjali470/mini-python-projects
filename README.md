# Mini Python Projects

This repository contains small Python projects built while learning programming concepts and problem-solving.

## Projects Included

### 1. Calculator Program
- Performs basic arithmetic operations
- Supports addition, subtraction, multiplication, division, etc.

### 2. Number Guessing Game
- User tries to guess a randomly generated number
- Demonstrates loops, conditions, and randomness

### 3. Password Validator
- Validates password based on:
  - Minimum length
  - Uppercase and lowercase letters
  - Digits
  - Special characters

## Concepts Used
- Functions
- Loops and conditionals
- Input validation
- String handling
- Random module
